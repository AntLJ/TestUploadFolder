■useful_headers
このディレクトリには有用なヘッダをまとめる。

ヘッダのみに限定し、リンクしなくても使えるようにすることで
利便性の向上を図っている。
ヘッダのみで実装するのが適当でないものについては
別途ライブラリを作成することになる。

他のライブラリからも使われることを想定する。
よって、循環参照を避ける為に、使っても良いヘッダは、
・C++標準ヘッダ
・boostのヘッダ
・atlbase.h
のみとする。

名前空間は uh とする。
内部にネストした名前空間を定義しても良い。

このディレクトリに配置するファイルはユニットテストを必須とする。
